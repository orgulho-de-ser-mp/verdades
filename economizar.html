Por que o órgão simplesmente não economiza parte do orçamento de um ano para usar no seguinte?"

É importante esclarecer um ponto: às vezes se questiona "por que o órgão simplesmente não economiza parte do orçamento de um ano para usar no seguinte?".

(...) é ilegal deixar de executar (gastar) intencionalmente parte do orçamento para carregar sobras para o ano seguinte.

Ora, é ilegal deixar de executar (gastar) intencionalmente parte do orçamento para carregar sobras para o ano seguinte. Existe uma vedação expressa no Art. 168, §2º da Constituição Federal, que proíbe que os recursos de um ano sejam transferidos para o orçamento do ano seguinte por vontade do gestor.

Ou seja, se o dinheiro foi destinado para o MPRS em determinado ano, ele deve ser usado para cumprir sua finalidade nesse ano; não podemos "segurar" recursos de propósito para acumulá-los. Aliás, na verdade, esse valor é deduzido das parcelas do duodécimo até que haja a compensação dos valores. Essa regra evita manobras e garante transparência no uso do dinheiro público dentro do período autorizado.
